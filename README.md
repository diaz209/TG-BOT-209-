Телеграм-бот для подсчета калорий, для отслеживания прогресса по диете/питанию. Считает твои Калории, Белки, Жиры, Углеводы. 

Используется SQLite база данных.

Хранятся: База создаётся автоматически при первом запуске, История веса, История питания, Цель калорий, ID пользователя.


| Команда        | Описание                         |
| -------------- | -------------------------------- |
| `/start`       | Запуск бота                      |
| `/help`        | Список команд                    |
| `/add_food`    | Добавить продукт и получить КБЖУ |
| `/my_calories` | Калории за сегодня               |
| `/set_goal`    | Установить цель по калориям      |
| `/add_weight`  | Записать текущий вес             |
| `/profile`     | Просмотр профиля                 |
| `/bmi`         | Расчёт индекса массы тела        |
